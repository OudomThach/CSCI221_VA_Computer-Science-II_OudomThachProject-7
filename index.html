Web Version

  1. Type a 24-hour time in the input box above (e.g. 13:07).
  2. Click Convert Time or press Enter.
  3. The 12-hour result appears below the button.
  4. Try invalid inputs like 10:65 or $$:87 to see error handling.
  5. All conversions are logged in the Conversion Log terminal.

C++ Version

  1. Open a terminal and navigate to the project folder.
  2. Compile: g++ -o TimeConverter TimeConverter.cpp -std=c++11
  3. Run: ./TimeConverter (Mac/Linux) or TimeConverter.exe (Windows).
  4. Enter a time when prompted (e.g. 13:07).
  5. Type y to convert another time, or n to quit.
